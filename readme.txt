Price Prediction Model - Mid Project Regression

Aim:
You are working as an analyst for a real estate company "Princess Peach Real Estate". 
Your company wants to build a machine learning model to predict the selling prices of houses based on a variety of 
features on which the value of the house is evaluated.

Used tools:

The tools were Python, MySQL, and for presentation Tableau and Slides. The raw data file has been imported and wrangled in 
Jupyter Lab notebook using Pandas, clean .csv files exported & imported & joined in Tableau for visualizations. 

The results for the prediction model were on average acceptable while some points would require more data cleaning/normalization

Challenges:

- removing outliers, bcz many categories consisted of outliers and needed to be handled individually

how to use & install:

- Fork & clone the entire repo on your hard-drive
- Run MySQL and open SQL file from the cloned folder
- Run the Jupyter notebook (.ipynb) in your preferred Python IDE
